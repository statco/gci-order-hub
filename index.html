GCI Order Hub LIVE

Serverless order automation — Shopify → Canada Tire / CJ Dropshipping / Walmart

Shopify Webhook Endpoints

Endpoint Description
POST/api/order-router Receives orders/paid webhook; routes TIRE- items to Canada Tire PO and NUPROZ- items to CJ Dropshipping. Sends Telegram + email notifications.
GETPOST/api/authorize-order HMAC-signed authorize link clicked from notifications. Confirms TIRE POs or auto-submits NUPROZ orders to CJ Dropshipping.

Walmart Marketplace

Endpoint Description
GETPOST/api/walmart-sync Syncs TIRE- prices and inventory from Shopify to Walmart. Runs daily at 09:00 UTC via cron. Supports ?dry=true and ?mode=listed.
GETPOST/api/walmart-order-sync Polls Walmart for new Created orders, acknowledges them, logs to Google Sheet, and sends Telegram alert. Runs every 15 min via cron.
POST/api/walmart-ship Marks a Walmart order as shipped. Accepts orderId, trackingNumber, carrier. Updates Google Sheet and sends Telegram confirmation.
GETPOST/api/walmart-item-feed Submits a bulk MP_ITEM_INTL feed to list TIRE- products on Walmart Canada. Returns a feedId to poll.
GET/api/walmart-feed-status Checks the processing status of a previously submitted Walmart feed. Requires ?feedId=<id>.
GET/api/getPendingOrders Returns all Walmart orders with status PENDING_CT from the Google Sheet, grouped by order ID.

CT Tracking

Endpoint Description
POST/api/ct-tracking-parser Parses a Canada Tire shipping confirmation PDF (base64 or multipart) and extracts tracking numbers and carrier info.
